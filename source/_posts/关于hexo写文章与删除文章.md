hexo 添加ico



#### hexo写文章命令：

1:进入hexo文件夹，鼠标右击，点击Git bash Here,进入git命令行
<!--more-->
2：添加文章命令：
   ```
   $ hexo new "关于hexo写文章与删除文章"
```
    此时在source\_posts目录下会生成一个.md文件
3：本地查看阅览
```
$ hexo s
```
4:推送到远端
```
hexo g -d
```
hexo生成文件并推送到远端


### 删除文章
1：删除db
2：删除xx.md
3:hexo clean
4:hexo g 
5:hexo d
