git的基础命令

Posted on 2019-11-11

git help
git clone git仓库地址 clone一个仓库到一个新的目录下
git init 创建一个空的git仓库或者初始化一个现有的git仓库
git add 将file文件内容添加到暂存区
git add . 将所有修改的文件添加进暂存区
git commit -m <提交说明> 将所有修改文件提交到当前分支
git rm 从暂存区里面删除file文件

git merge dev 把dev分支的代码合并到当前分支

git分支命令
查看、创建、切换和删除
git branch 查看本地分支
git branch –remote 查看远程分支
git branch -a 查看所有分支，包括本地和远程，对比远程存在本地那些没有的分支
git branch -v 查看各分支最后提交信息
git branch -m old_branch new_branch 修改本地分支名

git branch 分支名 创建本地分支
git checkout 分支名 切换本地分支
git checkout -b 分支名 创建本地分支并切换到该分支下
git branch -d 分支名 删除本地分支
git branch -D 分支名 强制删除本地分支

git push origin 本地分支名:远程分支名 创建远程分支（本地分支一定要存在）
git push origin -d f分支名 删除远程分支

git remote -v 查看远程仓库地址
git fetch 将某个远程主机的更新，全部取回本地
